## 概要

(6, 12)-Lennard Jones ポテンシャルを使用して、4 体系の遷移状態における形状安定性を研究するリポジトリです。
平衡点は 5 つ見つかっています。そのうち、真に安定な点は正四面体で配置する方法ですが、次に安定なのは菱形(rhombus)配置で、
これは 1 本の不安定方向と 5 本の安定方向、6 本のゼロモードを持ちます。摂動を与えて時間発展させると、より安定な形状である正四面体配置に崩壊するのですが、
安定方向に初期の振動を与えることで、崩壊までの時間を長くすることができることが分かりました。(scan_mode_collapse.py)

- src/: ソースコード
- reports/: 研究結果のまとめ
- slides/: プレゼンテーション資料
- results/: 計算結果
- references/: 参考文献

## 使用方法

1. uv でコードを実行できます。引数はソースコードを参照してください。

```bash
uv run src/lj4_3d.py
```

2. 計算結果ファイルは results/ ディレクトリに保存してください。ディレクトリの使い方は割と自由です。
3. 明らかになった事実があった場合、 reports/ ディレクトリに Markdown 形式でまとめてください。
