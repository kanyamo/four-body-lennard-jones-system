## 概要

(6, 12)-Lennard Jones ポテンシャルを使用して、4 体系の遷移状態における形状安定性を研究するリポジトリです。遷移状態はポテンシャルの鞍点であるため、摂動を与えるとより安定な形状

- src/: ソースコード
- reports/: 研究結果のまとめ
- slides/: プレゼンテーション資料
- results/: 計算結果
- references/: 参考文献

## 使用方法

1. uv でコードを実行できます。引数はソースコードを参照してください。

```bash
uv run src/lj4_3d.py
```

2. 計算結果ファイルは results/ ディレクトリに保存してください。ディレクトリの使い方は割と自由です。
3. 明らかになった事実があった場合、 reports/ ディレクトリに Markdown 形式でまとめてください。
